Welcome to ServJam Summer 2021

ServJam Summer 2021 is a modding event that will run from Saturday, August 14th through Sunday, August 27th. ServJam is a fun event to push the limits of server-side modding and a great excuse to make that new mod you've been thinking about, practice your server-side modding abilities, or try something new.

Requirements

  • Runs on Fabric for Minecraft 1.17.1
  • Cannot cause massive incompatibility with other entries
    • Avoid @Overwrite and try not to break core gameplay
  • Mods must be started during the time period
  • Must add gameplay, cannot be an admin utility mod
  • The mod's gameplay must work when a user joins the server with a vanilla client using the vanilla resourcepack
    • However, you can have an optional client mod or resourcepack if you'd like
  • Libraries and assets used cannot have been made for the purpose of ServJam
  • All code must be open, including used libraries
  • You will have 5 patch submissions to fix bugs or crashes, if it persists after the 5th submission, it will be disqualified
  • Be nice, don't make anything offensive or to harass someone

Theme

Anything! As long as it adds a feature to the game that anyone in survival can use, it works.

Submit your mod

Send the relevant information in the #submissions channel on the Discord server.
